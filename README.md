# PROJECT

Este repositorio contiene un entorno backend basado en **Java** y **Spring Boot**.

## Requisitos

- Java 17 o superior
- Maven (se utiliza `mvn` o el wrapper `./mvnw` si está disponible)

## Compilar y probar

```bash
mvn test
```

## Ejecutar la aplicación

```bash
mvn spring-boot:run
```

La aplicación de ejemplo carga un contexto vacío y se puede ampliar con controladores y servicios según sea necesario.
